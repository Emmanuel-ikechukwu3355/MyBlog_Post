# Django Blog Application

## Project Description
This is a simple Blog application built with Django.  
It demonstrates full **CRUD** operations — Create, Read, Update, and Delete — for blog posts, as well as adding comments to each post.  

This project was built to practice Django models, ModelForms, and CRUD functionality using function-based views.


## Features
- Create, view, edit, and delete blog posts  
- Add comments to posts  
- Each page has its own clean template and basic styling  
- Built with Django 5+ and Python 3
