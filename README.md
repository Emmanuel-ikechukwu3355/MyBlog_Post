* Django Blog_post Project

This is a simple blog project built with Django.  
It supports creating, reading, updating, and deleting blog posts, as well as submitting feedback.

* Features
- Blog post CRUD operations (Create, Read, Update, Delete)
- Feedback form with name, email, and message
- Admin panel for managing posts and feedback
